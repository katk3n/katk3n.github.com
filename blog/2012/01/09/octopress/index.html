Octopress を使ってみる

January 09, 2012

Tumblr を使って自分が勉強した事のメモとかを書いていましたが、 Octopress という技術ブログ向けフレームワークがあるということなので、使ってみました。

http://octopress.org/

インストール

以下を参考にしながらインストール。 http://tokkonopapa.github.com/blog/2011/12/30/octopress-on-github-and-bitbucket/

新規投稿

rake new_post['title']

なお、自分の環境だとファイルが作られたときのフォーマットが iso-2022-jp になっているようなので、初回は # In vim :set fileencoding=utf-8 で文字コードを直す。

デプロイ

# ページの作成
rake generate
# プレビュー
rake preview
# デプロイ
rake deploy
# 作成とデプロイを同時にやる場合
rake_gendeploy

markdown 記法

記事は markdown 記法と呼ばれる記法で書くらしい。

参考：
http://www.kotono8.com/wiki/Markdown#.E8.A6.8B.E5.87.BA.E3.81.97 http://nakama.cc.sophia.ac.jp/help.php?file=advanced_markdown.html

見た目の変更とかは後々やろうと思います。
とりあえず、前に書いた記事のうちいくつかを移してみよう。
