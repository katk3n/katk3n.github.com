FreeBSD のセットアップ

January 26, 2012

FreeBSD 9.0 のインストールとか設定とか

FreeBSD 9.0 のインストール

FreeBSD 9.x 以降では bsdinstall と呼ばれるインストールプログラムを使う。
以下を見ながらインストール
http://www.freebsd.org/doc/ja_JP.eucJP/books/handbook/bsdinstall.html

ホスト名の変更

$ hostname "ホスト名"

で一時的に変更できるが、OS を再起動すると元に戻ってしまうため、
/etc/hosts, /etc/resolv.conf, /etc/rc.conf の設定も直しておく (Linux の場合は /etc/sysconfig/network)

ports からパッケージのインストール

ports ツリーの構成

$ pkg_add -r portsnap  # portsnap が無い場合まず package からインストールする
$ portsnap fetch
$ portsnap extract
$ portsnap update

ports にあるパッケージは /usr/ports/ 以下にカテゴリ別に置かれている。
http://www.freebsd.org/cgi/ports.cgi
からパッケージの検索ができる。
また、 $ whereis でどこのパッケージがあるか探すことができる。

sudo

FreeBSD には デフォルトで sudo は入っていない $ su root # cd /usr/ports/security/sudo # make install clean インストール後 visudo で sudoers を編集

portaudit

パッケージのセキュリティに関する脆弱性を調べる $ cd /usr/ports/ports-mgmt/portaudit $ sudo make install clean $ portaudit -F

portupgrade

portinstall, portupgrade 等が使えるようになる $ cd /usr/ports/ports-mgmt/portsupgrade $ sudo make install clean

ports package の設定をやり直す

$ sudo make config

ports package のアンインストール

$ sudo make deinstall

ports package の再インストール

$ sudo make reinstall

login 時の shell の設定

$ chsh -s /usr/local/bin/zsh

httpd 起動のための設定

accept filter を使うために accf_http.ko というカーネルモジュールを読み込む必要がある。 $ sudo kldload accf_http.ko OS 起動時に読み込ませるには、/boot/loader.conf に accf_http_load=“YES” を追加する。
