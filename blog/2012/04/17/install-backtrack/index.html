BackTrack 5 RC2 をインストール

Apr 17th, 2012 | Comments

ペネトレーションテストに特化した Linux ディストリビューションの BackTrack をインストールしました。

http://www.backtrack-linux.org/

BackTrack とは

  • Ubuntu がベース (BackTrack4 以降)
  • 様々なセキュリティツールが入っている

インストール

Virtual Box にインストールした。

  • OS タイプは Ubuntu 64 bit
  • NAT + ホストオンリーアダプタ

起動したら root/toor で root にログインして、

$ startx

で GUI を起動する。

設定

ユーザの追加

$ adduser <username>

useradd だとホームディレクトリの作成や設定のコピー等をやってくれない

sudo の設定

$ visudo

エディタは vi でないので注意。

%<username> ALL=(ALL) ALL

を追加。

host 名の変更

$ sudo vi /etc/hostname
$ sudo vi /etc/hosts

IP の設定

eth1 の部分を以下のように編集

ssh の設定

$ sudo apt-get --purge remove openssh-server
$ sudo apt-get install openssh-server

sshd 再起動

$ sudo /etc/init.d/ssh restart
$ sudo lsof -nPi:22  # 確認

鍵の設定が終わったら、PasswordAuthentication を no に戻しておく。
