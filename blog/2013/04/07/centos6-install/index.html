インストール

Time Zone Selection

“System clock uses UTC” にチェックを入れ、”Asia/Tokyo” を選択し、OK

Partitioning Type

iptables, SELinux 無効化

/boot/grub/grub.conf の kernel 行に enforcing=0 を追記する。
